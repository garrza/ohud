# Orbital HUD

A spacecraft-themed watch face for Garmin (currently only for fenix 7).

![Orbital HUD](ohud-clip.gif)

**[Download on the Connect IQ Store](https://apps.garmin.com/apps/9bafd9ee-a64b-49ed-8048-3638442c8a8e)**

## Features

- Animated rotating planet with orbital ring
- Rocket seconds indicator orbiting the watch bezel
- Live heart rate sparkline from real HR data
- 16 configurable data fields: body battery, calories, floors, distance, active minutes, altitude, steps, weather, pressure, respiration, VO2 max, HRV, sleep, recovery, and more
- Sunrise/sunset times
- HUD-style corner brackets, scan lines, and dashed separators

## Themes

6 color themes that tint all UI elements and the planet animation:

- **Deep Space** — Cyan
- **Phosphor Green** — Green
- **White Minimal** — White
- **Red Tactical** — Red
- **Amber Retro** — Amber
- **Solar Flare** — Orange

## Settings

Configurable via the Garmin Connect app:

- Color theme
- Date format (standard or stardate)
- Toggle individual data fields on/off
